Restaurant Menu & Order MERN App (sample)

Structure:
- backend: Express + Mongoose API
- frontend: React app

Quick start (local):
1. Backend:
   - cd backend
   - copy .env.example to .env and set MONGO_URI
   - npm install
   - npm run dev  (or npm start)
2. Frontend:
   - cd frontend
   - copy .env.example to .env (set REACT_APP_API_URL if needed)
   - npm install
   - npm start

This zip contains minimal, workable example code for a small demo app.
